Detailed Description

template<typename T, typename TComponent = T, bool Optional = false, typename TDynamicStorage = buf::DynamicSSBO<T>, typename TStaticStorage = buf::StaticSSBO<T>>
class bl::rc::ds::ObjectStorageBuffer< T, TComponent, Optional, TDynamicStorage, TStaticStorage >

Generic descriptor set binding providing ECS entity components to shaders.

Template Parameters
TThe shader format of the data to provide
TComponentThe ECS component to populate from
OptionalWhether or not this binding is optional for objects
TDynamicStorageThe type of buffer to use for dynamic entities
TStaticStorageThe type of buffer to use for static entities
