Detailed Description

template<typename T>
class bl::util::IdAllocator< T >

Utility to allocate integral ids in some range [0, max). Allows ids to be reused as well as tracks the current highest id to allow for tighter iteration over the id range. Ids are allocated in linear order.

Template Parameters
TThe integral type to use for ids
