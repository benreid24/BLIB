Detailed Description

template<typename T>
class bl::container::DynamicObjectPool< T >

Basic object pool that may vary in size. Similar to FastEraseVector in that erase is O(1), however objects in the pool maintain their indices in the DynamicObjectPool. Object slots are marked free on erase(), reused on add(), and the pool grows in size if no slot is when add() is called. Objects must implement a default constructor and copy constructor.

Template Parameters
TType of object to store. Works best with small POD objects
