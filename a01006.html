Detailed Description

template<typename... TEvents>
class bl::event::ClassGuard< TEvents >

Version of scope guard but for classes. Classes that subscribe to event buses through a member ClassGuard will automatically unsubscribe when they are destructed. The catch is that they may only subscribe to a single event bus at a time.
