Detailed Description

template<std::uint16_t Id, typename C>
struct bl::serial::binary::SerializableFloatField< Id, C >

Descriptor class for a float member of a class to be serialized.

Template Parameters
IdThe id of the data member. Allows formats to change without invalidating existing data
CThe type of the class that the field belongs to
FThe type of the float data member
