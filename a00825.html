Detailed Description

template<typename T>
class bl::container::ObjectWrapper< T >

Simple wrapper around single objects allowing for delayed construction, memory reuse, and destruction at any time. Note that space is allocated inline with the object and will therefor be wherever ObjectWrapper is allocated (stack or heap). Because of this, moving from another ObjectWrapper will invoke T's move constructor instead of swapping memory.
