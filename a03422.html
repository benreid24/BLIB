Detailed Description

template<typename T>
class bl::ctr::ObjectWrapper< T >

Thin wrapper over objects that functions similar to std::optional but does not have any overhead. Does not store the presence state of the object contained. That must be done by the user. Intended to be used within other data structures that can know if elements are constructed or not.

TThe type of object to wrap
