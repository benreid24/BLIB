Detailed Description

template<typename TCom, typename TSys = sys::DrawableSystem<TCom>>
class bl::gfx::Drawable< TCom, TSys >

Base class for Drawable objects. A Drawable object should inherit this class for the drawable component that it wraps.

Template Parameters
TComThe drawable component type (ie Sprite)
TSysThe drawable system. Must implement DrawableSystem interface
