Detailed Description

template<unsigned int MovementControl, unsigned int ActivateTrigger>
class bl::menu::MovementDriver< MovementControl, ActivateTrigger >

Helper class to receive user input from the InputSystem and drive a menu with it. Must be subscribed to an input::Actor to work.

Template Parameters
MovementControlThe control index of the MovementControl to move on
ActivateTriggerThe control index of the Trigger control to activate on
