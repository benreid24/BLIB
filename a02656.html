Detailed Description

template<typename T>
class bl::ctr::RingBuffer< T >

Simple circular buffer implementation for small objects. Underlying storage is a std::vector of buffers. Objects are constructed with placement new and destructors are called when erased, cleared, or overwritten.

TType of object to store
