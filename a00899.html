void bl::engine::Engine::replaceState ( State::Ptr  next)

Sets the next state for the following engine update loop. May be called at any time, the next state will not be transitioned to until the start of main loop. This replaces the current state so that it will not be returned to when the next state is popped. Equivalent to setting PopState flag and pushing state together.
