Detailed Description

template<typename T>
class bl::pcl::ParticleManager< T >

Particle manager for a specific type of particle. Instances are owned by the ParticleSystem and should be obtained from there.

Template Parameters
TThe type of particle to manage
TRendererThe type of renderer plugin to use
