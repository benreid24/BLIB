Detailed Description

template<typename TCoord, typename TPayload>
class bl::ctr::QuadTree< TCoord, TPayload >

Fairly basic implementation of a quadtree for spatially partitioning objects. Works most efficiently if the payload type is trivially movable. Stored objects are keyed on position.

Template Parameters
TCoordThe type of coordinate to use. Must contain members x and y and define operator ==
TPayloadThe type of payload to store
