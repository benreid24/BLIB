Detailed Description

template<typename TKey, typename TValue>
class bl::ctr::IndexMappedList< TKey, TValue >

Specialized storage that is like a multimap, but internally is comprised of only a few vectors. This keeps the storage relatively cache friendly and helps to minimize fragmentation over time. Chief usage is one-to-many relationship mapping in the ECS.

Template Parameters
TKeyThe key type. Must be an unsigned integer
TValueThe value type
