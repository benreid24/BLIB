Detailed Description

template<typename T, std::size_t Capacity>
class bl::ctr::StaticRingBuffer< T, Capacity >

Simple circular buffer implementation for small objects. Underlying storage is a std::array of buffers. Objects are constructed with placement new and destructors are called when erased, cleared, or overwritten.

Template Parameters
TType of object to store
CapacityThe max capacity of the ring buffer
