Calculates the shortest path between the two points using the map callback. Map callback signature should be void(const TNode&, std::vector<std::pair<TNode, int>>&). The vector should be populated with nodes adjacent to the given node along with the cost to move to those nodes from the given node. Distance estimator signature should be int(const TNode& from, const TNode& to)

Template Parameters
TMapCallbackCallback to get adjacent nodes and their movement cost
TDistFuncCallback to estimate distance between two non-adjacent nodes
Parameters
startThe point to start at
destinationThe point to end at
adjacentNodeCbCallback to get travel nodes for a given node
distCbCallback to estimate distance between two non-adjacent nodes
resultThe vector to store the calculated path in
Returns
True if a path was found, false if a path was not found
