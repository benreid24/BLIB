template<typename TBuffer, std::uint32_t DefaultCapacity>
class bl::rc::sr::BufferShaderResource< TBuffer, DefaultCapacity >

Lightweight shader input wrapping a buffer that can be used for uniforms or SSBOs.

Template Parameters
