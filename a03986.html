Detailed Description

template<typename T>
class bl::rc::buf::FullyDynamicSSBO< T >

Device-local SSBO intended for use with per-object data for dynamic objects. This version of DynamicSSBO does not provide a CPU-side buffer and assumes that the user will fully rewrite the entire buffer every frame. If that is not the case then use DynamicSSBO.

Template Parameters
TThe type of data to pack into the SSBO
