Detailed Description

template<typename T, util::TemplateString Name, bdl::AssetBundleConfig BundleConfig = bdl::AssetBundleConfig()>
class bl::as::DriverGeneric< T, Name, BundleConfig >

Generic driver that can be used for assets that are trivially serializable.

Template Parameters
TThe type of data stored in the payload. Must be serializable
NameThe string literal name of this driver type
BundleConfigThe bundle config for assets of this type
