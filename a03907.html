Determines the behavior when sourcePath is a directory.

Enumerator
BundleAllFiles 

All files are placed in the same bundle.

CreateBundleForEachContained 

Each contained file/folder gets its own bundle.

CreateBundleForEachContainedRecursive 

Every file gets its own bundle, including files in folders.
