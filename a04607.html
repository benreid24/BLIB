Detailed Description

template<typename T, typename GpuT>
class bl::pcl::DescriptorSetFactory< T, GpuT >

Descriptor set factory for the default descriptor set used by the default renderer. Provides an array of particle structs as well as a singular global object of config data to the shaders. Types must be specified in a specialization of RenderConfigMap.

Template Parameters
TThe particle type
GpuTThe particle type to use in the GPU-visible buffers. Must be assignable from T
