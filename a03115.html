Detailed Description

template<typename T>
class bl::rc::vk::PerSwapFrame< T >

Utility class to manage objects where one instance is needed per image in the swap chain. Provides init and cleanup helpers as well as an accessor for the current object based on the current swap chain index.

Template Parameters
TThe type of object to store per swap frame. Must be default-constructible
