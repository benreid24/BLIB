Detailed Description

template<typename T>
class bl::gfx::vk::PerFrame< T >

Utility struct to easily double-buffer objects and data on a per-frame basis. One object of type T will exists per concurrent frame and the current will be accessible.

Template Parameters
TThe type of object to buffer per-frame
