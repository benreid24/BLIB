Detailed Description

template<typename TResourceType>
class bl::resource::Ref< TResourceType >

Represents a reference to a managed resource. Performs reference counting, but resources are never deallocated by this, only by ResourceManager.

Template Parameters
TResourceTypeThe resource type being referenced
