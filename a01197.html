Detailed Description

template<typename T>
class bl::container::RingQueue< T >

Queue that uses a fixed size buffer internally to maintain contiguous storage and avoid allocations. The queue cannot hold more elements than it's capacity.

Template Parameters
TThe type of object to hold in the queue
