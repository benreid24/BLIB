Detailed Description

template<typename T>
class bl::rc::buf::UniformBuffer< T >

Device-local, staged, uniform buffer array. Intended for small, dynamic data such as per-frame scene data. Not intended for larger data sets such as per-object data. Writes go directly to the staging buffer, so the used portions of the UBO must be written to every frame otherwise stale data will be copied to the GPU.

Template Parameters
TThe type of data to pack into the UBO
