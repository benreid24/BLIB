Detailed Description

template<typename T>
class bl::ecs::trait::ChildAware< T >

Component trait for components that are parent-able. Parent is set in the Registry as an entity-level relationship. Components should inherit this trait and pass their own type as the type parameter T. This trait provides a vector of child pointers on the component.
