Detailed Description

template<typename... TComponents>
class bl::ecs::ComponentSet< TComponents >

Contains a set of components for a single entity. May not always be valid. Does not update if components are added or removed from the entity. Sets from views are always valid.

Template Parameters
TComponentsThe components to fetch for the entity
