Detailed Description

template<typename T>
struct bl::pcl::RenderComponentMap< T >

Compile-time config map used by the default Renderer<T> implementation. By default EcsComponent is used to render the particle system. Specialize this struct to specify a custom component to use instead. The constructor must have the same signature as the EcsComponent constructor.

Template Parameters
TThe particle type to map to the renderable component
