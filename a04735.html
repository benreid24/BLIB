Detailed Description

template<typename T, Alignment Align, VkBufferUsageFlags Usage, VkMemoryPropertyFlags MemoryPool, VmaAllocationCreateFlags AllocFlags = 0, VkMemoryPropertyFlags FallbackMemoryPool = MemoryPool>
class bl::rc::buf::base::BindableBufferSingleSourced< T, Align, Usage, MemoryPool, AllocFlags, FallbackMemoryPool >

Single buffered buffer class that provides buffer management and direct access for mapped buffers that are CPU visible. Also provides a host buffer to store the contained payloads.

Template Parameters
TThe payload type that the buffer will hold
AlignThe alignment to use if storing more than one element
UsageThe usage type to create the buffers with
MemoryPoolThe type of memory to allocate the buffer in
AllocFlagsAllocation flags to forward to VMA
FallbackMemoryPoolMemory pool to use if primary allocation fails
