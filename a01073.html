Detailed Description

template<typename T>
class bl::container::ObjectWrapper< T >

Thin wrapper over objects that functions similar to std::optional but does not have any overhead. Does not store the presence state of the object contained. That must be done by the user. Intended to be used within other datastructures that can know if elements are constructed or not.
