Detailed Description

template<typename T, typename TStorage = std::vector<T>>
class bl::util::VectorRef< T, TStorage >

Utility struct to provide a stable iterator into a vector where indices do not change.

Template Parameters
TThe type contained within the vector to index into
TStorageThe storage type. Defaults to std::vector. Only requires operator[]
