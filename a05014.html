template<typename TShaderResource, sr::StoreKey StoreKey, std::uint32_t RenderPassId, RenderPassBehavior RenderPassMode, DepthAttachmentType DepthAttachment>
class bl::rc::rgi::GenericTargetAsset< TShaderResource, StoreKey, RenderPassId, RenderPassMode, DepthAttachment >

Generic asset for a templated set of attachments.

Parameters
TShaderResourceThe type of shader resource that provides the attachments
StoreKeyThe shader resource store to get the shader resource from
Template Parameters
RenderPassIdThe id of the render pass to use
AttachmentCountThe number of attachments to create, excluding depth & resolve
RenderPassModeWhether this asset is responsible for render pass start/stop
