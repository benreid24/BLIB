Detailed Description

Data class for 2d animations. Provides the backing for slideshows and skeletal animations. Multiple AnimationData objects can be combined to create a stateful animation.

bool bl::gfx::a2d::AnimationData::addState ( std::size_t  stateIndex,
const AnimationData &  stateAnim,
bool  enforceSlideshow = true 
)

Combines the stateAnim into this one and updates the next frame lookups. May fail if the animation is not compatible with this one. Compatible animations have the same number of shards per frame. Same spritesheet must also be used. State 0 is always this base animation.

Parameters
stateIndexThe 0-based index of the state that the given animation is for
stateAnimThe animation to combine into this one
enforceSlideshowTrue to also validate that slideshow status is the same or better
Returns
True if the animation could be combined, false on error
