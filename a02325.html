Detailed Description

Component used to scale entities in overlays according to fixed rules. sys::Overlay scaler ensures that scales are up-to-date as the overlay changes size. Entity transforms should not be manually scaled if an OverlayScaler is present and active.

◆ ScissorMode

enum bl::com::OverlayScaler::ScissorMode

Represents the scissor behavior for an entity controlled by an overlay scaler.

Enumerator
ScissorInherit 

Default setting. Uses same scissor as parent, or Observer scissor for root elements.

ScissorSelf 

Sets the scissor to the bounds of the object.

ScissorSelfConstrained 

Sets the scissor to the bounds of the object and constrains against the parent scissor.

ScissorObserver 

Sets the scissor to the top-level observer scissor.
