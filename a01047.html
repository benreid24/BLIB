Detailed Description

template<std::uint16_t Id, typename T>
class bl::file::binary::SerializableField< Id, T >

Template class that represents a serializable field for File use. Strings and integral types are supported out of the box, as well as SerializableObjects. User defined types must either implement SerializableObject or a custom version of Serializer.

Template Parameters
IdThe id of this field. Must be unique within a SerializableObject
TThe type of the field to be serialized
