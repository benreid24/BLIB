Detailed Description

template<typename TResourceType>
struct bl::resource::AsyncLoader< TResourceType >

Specialized loader meant for loading resources into a Manager in the background. Default implementation simply calls load() on the manager, but custom background loaders can be built for large resources and should leverage Manager::add() to insert constructed resources.

Estimate the relative amount of work required for the given resource. The number itself is arbitary and the value does not matter. It must simply be on the same scale as the other numbers returned by this method for different resources. The default behavior returns the file size, in bytes, divided by 10,000.
