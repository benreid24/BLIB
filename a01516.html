Detailed Description

template<std::uint16_t Id, typename C, typename T>
struct bl::serial::binary::SerializableField< Id, C, T >

Descriptor class for a data member of a class to be serialized.

Template Parameters
IdThe id of the data member. Allows formats to change without invalidating existing data
CThe type of the class that the field belongs to
TThe type of the data member
OffsetThe offset of the data member in it's class
