Detailed Description

template<typename T, std::uint32_t N>
class bl::rc::rg::MultiAsset< T, N >

Helper class that allows an asset to be duplicated N times while being treated as a single asset by the render graph.

Template Parameters
TThe type of asset to duplicate
NThe number of duplicates to make
