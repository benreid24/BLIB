Detailed Description

template<typename T>
struct bl::pcl::RenderConfigMap< T >

Compile-time mapping struct for use with the default Renderer<T> implementation. Specialize this for your particle type and provide the pipeline and transparency settings to render with.

Template Parameters
TThe particle type to specify render settings for
